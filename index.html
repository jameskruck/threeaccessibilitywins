About These Examples

This collection of pages demonstrates real accessibility issues that instructional designers, educators, and content creators encounter daily. Each example is based on actual workplace scenarios and shows both the problems and the solutions.

How to Use During Your Workshop

  • Live Demonstrations: Navigate between pages to show immediate before/after comparisons
  • Screen Reader Testing: Use assistive technology with these examples to demonstrate the actual user experience
  • Tool Practice: Run WAVE, contrast checkers, and other accessibility tools on these pages
  • Group Activities: Have participants identify issues before revealing the solutions

What Each Section Covers

Tables (Pages 1-2)

  • Proper use of table headers (<th> vs <td>)
  • Scope attributes for complex tables
  • Table captions and summaries
  • Screen reader navigation patterns

Content Structure (Pages 3-4)

  • Heading hierarchy and semantic markup
  • Color contrast requirements
  • Alternative text for images
  • Accessible color usage beyond contrast

Alt Text Workshop (Page 5)

  • Real examples from workplace documents
  • Expert feedback on common problems
  • Interactive before/after comparisons
  • Best practices for different image types

Technical Notes

These pages are designed to work offline and are compatible with all major screen readers and accessibility testing tools. They use semantic HTML and follow WCAG 2.1 AA guidelines themselves, serving as positive examples of accessible web design.
